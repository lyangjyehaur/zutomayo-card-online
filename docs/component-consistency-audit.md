## 推薦抽象優先順序

1. `Dialog/Sheet/IconButton/Button`：影響 a11y、touch target、modal 一致性，收益最高。
2. `BoardLayout` + `CardDetailSurface`：Battle 是核心場景，且目前樣式最容易互相覆蓋。
3. `ResponsiveToolbar` + `DataList`：Feedback/Admin/I18n/Deck Builder 可共用，能減少大量 page CSS。
4. `Alert/EmptyState/LoadingState/Toast`：降低狀態顯示分散與文字風格不一致。
5. `PageShell/PageHeader` variants：最後收斂頁面骨架與低高度/手機規則。
