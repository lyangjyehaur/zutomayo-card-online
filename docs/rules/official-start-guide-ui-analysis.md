# 官方 Start Guide 規則校準（UI 分析）

> 來源：https://zutomayocard.net/start-guide/（2026-07 讀取）
> 目的：作為線上對戰 UI 的規則單一來源之一；記錄舊實作/舊 UI 與官方的偏差與修正。
> 相關：`docs/uiux/rule-to-ui-mapping.md`、`docs/uiux/battlefield-online-adaptation.md`

## 1. 遊戲概要

2 人對戰；各用 20 張牌組；互相造成傷害，先把對手 HP 打到 0 者勝。核心特色：操縱遊戲內時間「クロノス」，卡牌在**夜/晝**時攻擊力與效果改變。

## 2. 牌組與初始準備

- 每人：20 張牌組 + HP 計數器；共用：場地墊 1、クロノスメダル 1。
- 構築限制：同 pack 同編號最多 2 張；建議 Character ≥ 50%。
- 準備：猜拳決定**夜側玩家** → メダル置於「真夜中」、HP=100 → 洗牌（實體有互換洗牌儀式，線上由伺服器洗牌代替）→ 各抽 5 張。

## 3. 卡牌種類與卡面資訊

- **Character**：有攻擊力（NIGHT/DAY 二値）與效果。
- **Enchant**：單回合的臨時效果。
- **Area Enchant**：跨回合持續效果（置於 Set Zone C）。
- 卡面：種類/卡名/屬性（闇・炎・電気・風・カオス）/時計値/攻擊力/效果/パワーコスト(PC)/SEND TO POWER/稀有度/收錄包/編號。

## 4. 官方場地區域

バトルゾーン、セットゾーン A/B/C、パワーチャージャー、デッキゾーン、アビス、HPメーター、クロノス。

## 5. Chronos 規則含義

- **18 刻度**錶盤（官方場地墊：夜/晝各 9 格平分）；**青（藍）＝夜、赤（紅）＝晝**；起始「真夜中」，順時針推進。
- Assumption: start-guide 頁面文字未寫明刻度數；18（9+9）依官方フィールドマット實物與使用者確認。舊實作為 12 刻度（夜 6/晝 6）— 屬**規則偏差**，已於引擎 CHRONOS_MAPPING 修正為 18。
- 每回合推進量 = 雙方本回合出牌時計値合計。

## 6. 夜/晝的影響

- メダル在夜 → 用 NIGHT 攻擊力；在晝 → 用 DAY。
- 效果處理從「メダル所在側」的玩家開始。

## 7. Battle Zone

雙方各 1 張對戰中的 Character；攻擊力比較低者受**差值**傷害；PC 不足時攻擊力＝0。

## 8. Set Zone A/B/C

- A/B：本回合從手牌設置的卡（蓋牌）；**上回合勝者只能設 1 張（A）、敗者可設 2 張（A+B）、平手各 1 張**。
- C：已發動的 Area Enchant 常駐位；替換時舊卡依 SEND TO POWER 去 Charger/Abyss；同回合設 2 張 Area Enchant 時 A 的進 C。

## 9. Power Charger

離場且**有** SEND TO POWER 的卡置於此；全部 SEND TO POWER 合計＝該玩家總パワー（PC 門檻資源）。

## 10. Deck Zone

20 張、背面朝上；回合結束抽「本回合出牌數」。

## 11. Abyss

離場且**無** SEND TO POWER 的卡置於此（不產生資源）。

## 12. HP Meter

初始 100；必須隨時可讀 — 直接連動勝敗。

## 13. 起始手牌與引き直し（Mulligan）

各抽 5 張；限一次：任選 N 張蓋置場邊 → 宣告 N → 抽 N 張 → 蓋置的牌回牌組洗勻。

## 14. 第 1 回合流程

雙方從手牌選 1 張**蓋牌置入 Battle Zone** → 同時喊「嫌（やぁ）」翻開 →（非 Character 立即送 Charger/Abyss）→ 時計推進 → 效果（メダル側先，PC 不足不發動）→ 攻擊力決定（夜/晝）→ 傷害計算 → 回合結束抽本回合出牌數。

## 15. 第 2 回合以後

①設置（勝者1/敗者2/平手各1，蓋牌至 A 或 A+B）→ ②同時公開 → ③時計推進 → ④Character 入替（新 Character 進 Battle Zone，舊卡依 STP 去向；設 2 張 Character 時 **A 優先**）→ ⑤Area Enchant 入替（進 C，同上 A 優先）→ ⑥效果處理（メダル側先）→ ⑦攻擊力決定 → ⑧傷害計算 → ⑨回合結束（A/B 剩卡依 STP 去 Charger/Abyss；抽本回合出牌數）。

## 16. 勝敗條件

- HP ≤ 0 → 對手勝（即時）。
- 回合結束抽牌時牌組不足 → 該玩家敗。

## 18. 新版 UI 必須修正/強調的規則語義（已落地）

1. Chronos 為戰場核心視覺；晝夜=官方紅/藍雙色（同時成為全站 Art Direction 的雙色系統 `--time-night`/`--time-day`）。
2. BattleZone 常駐顯示雙方生效攻擊力（晝夜著色、PC 不足顯示 0＋警示）— 傷害=攻擊力差的規則可視化。
3. SetZone A/B/C 各自標注用途（設置①/設置②・敗者/持續附魔），C 區虛線+青色與 A/B 區隔。
4. ChargeZone 常駐「總 Power」；AbyssZone 明示「無 STP 卡的去向」；DeckZone 張數常駐（敗北攸關）。
5. HP 於所有斷點常駐（PlayerStatus bar），警示分級（≤50 晝紅、≤25 danger）。

**需要人工確認的規則項**：無核心邏輯偏差需要改動；D9（構築建議提示）為功能建議非規則錯誤。
