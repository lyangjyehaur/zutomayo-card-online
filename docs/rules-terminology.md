# 規則術語 i18n 字典

本文件定義卡牌介面、教學、PostgreSQL 卡牌文本、官方 Q&A 與勘誤共同使用的 canonical 規則術語。涵蓋目前正式規則會使用的卡面標記、卡種、區域、屬性、對戰角色、流程階段、規則動作、牌組位置與勝負狀態；卡名、歌曲名及一般敘述不屬於本字典。程式可執行版本位於 `src/rulesTerminology.ts`；修改任一術語時，兩者必須同步更新。

## 保留標記

以下卡面或系統標記在所有衍生語言中保留原文與大小寫，不另造譯名：

- `Power`
- `Power Cost`
- `SEND TO POWER`
- `HP`

日文權威來源仍依官方日文顯示 `パワー`、`パワーコスト`、`クロノス`；英文標記 `SEND TO POWER` 保持不變。系統名稱 `Chronos` 在中文與英文保留原文，韓文使用音譯 `크로노스`。

## 維護規則

- 新增正式規則機制、階段、區域、卡種、Q&A 分類或卡面欄位時，必須先在本文件與 `src/rulesTerminology.ts` 增加六語 canonical 詞條，再新增或修改翻譯。
- 卡名不進本字典；Q&A 與勘誤一律從 PostgreSQL 的已複核 canonical 卡名解析。
- 一般文案可因語法調整詞形，但不得另造同義術語。需要保留的別名只能放入非 canonical 稽核規則，不能成為第二套譯名。
- 字典的「完整」以目前已實作且會公開顯示的正式規則概念為界；未實作、非官方或純 UI 操作詞不納入。

## 執行與稽核

- `src/i18n/*.ts` 的獨立 UI 標籤由 `src/__tests__/rulesTerminology.test.ts` 對照程式 glossary。
- `scripts/officialRulingsData.ts` 在五語裁定 release 前拒絕未本地化卡種／區域及 `Power Cost`、`SEND TO POWER` 的別名。
- `scripts/cardDerivedEffects.ts` 在衍生卡牌效果匯入 PostgreSQL 前執行相同檢查。
- 自動替換後必須人工抽查完整句子、韓文助詞、中文空格及原有段落換行；通過 glossary 不等於完成語言複核。
