## 專案用途

本專案：

- 是免費、無廣告、非營利、非商用的粉絲專案。
- 以推廣 ZUTOMAYO CARD 玩法及促進海外玩家參與為目的。
- 只在遊戲、牌組構築、卡牌瀏覽與規則說明所需範圍展示卡牌素材。
- 不販售卡圖、不提供素材包、批量下載或以素材本身獲利。
- 明確標示為非官方，且不宣稱與官方存在授權合作關係。

## Owner 決策

專案 Owner 決定依官方公開政策與素材公開方式進行非營利 Public Beta，不要求 Public Beta 前另行取得個別書面授權。

Owner 理解「向不特定多數人散布」對公開 Web 服務仍可能存在解釋空間，並接受這項剩餘風險，前提是持續遵守本文件列出的非營利、用途限制、聯絡與下架措施。

因此，IP 個別授權不再是 Public Beta hard blocker；RR-01 的剩餘工作是公開政策、聯絡管道、信箱收件與下架流程的驗證。

## 必須持續遵守的限制

- 不加入廣告、付費會員、付費牌組、付費素材、卡圖販售或其他直接營利。
- 不提供原始卡圖壓縮包、批量素材 API 或以繞過官方取得方式為目的的下載功能。
- 不冒充官方、不使用會使一般玩家誤認為官方服務的聲明。
- 保留可用的權利人聯絡信箱與處理記錄。
- 收到可信通知時，可先限制、替換或暫時隱藏相關素材，再完成確認。

## 必須重新開啟決策的情況

- 官方政策更新、撤回或新增與線上遊戲、卡牌素材直接相關的限制。
- 官方或相關權利人提出修改或下架要求。
- 專案計畫加入廣告、贊助、付費內容、募資回饋或其他營利方式。
- 專案新增素材包、批量下載、第三方素材 API 或與目前用途不同的再散布方式。
